# LSPatch UNITE Framework

# Introduction 

Rootless implementation of LSPosed framework, integrating Xposed API by inserting dex and so into the target APK.

## Supported Versions

- Min: Android 9
- Max: In theory, same with [LSPosed](https://github.com/JingMatrix/LSPosed#supported-versions)

## Download

For stable releases, please go to [Github Releases page](https://github.com/Lobanokivan11/LSPatch-UNITE/releases)
For canary build, please check [Github Actions](https://github.com/Lobanokivan11/LSPatch-UNITE/actions)
Note: debug builds are only available in Github Actions

## Usage

+ Through manager
1. Download and install `manager.apk` on an Android device
2. Follow the instructions of the manager app
